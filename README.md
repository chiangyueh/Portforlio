# 安裝與執行步驟
1.請先打開您的git並clone本專案至本地  
```bash
git clone https://github.com/chiangyueh/restaurantList.git
```  
2.切換目錄至專案資料夾  
```bash
cd Portfolio
```    
3.使用npm套件安裝相應模塊  
```bash
npm i
```  
4.啟動項目  
```bash
npm run dev
```  
本專案運行於3000端口:  
```bash
localhost:3000/
```
